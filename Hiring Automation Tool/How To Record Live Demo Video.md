# How To Record The Live Demo Video

Use this when you want to capture the actual running tool on your system and share it with the client.

## Before Recording

1. Close `Hiring_Automation_Candidates.xlsx`.
2. Open `Run Hiring Automation Tool.bat` or `Run_CV_Agent_UI.bat`.
3. Keep the mailbox password private.
4. Do not show full email IDs or phone numbers in Excel during the recording.

## Suggested Recording Flow

1. Show the title:

```text
Hiring Automation Tool - Motifzone Pvt Ltd
```

2. Open the tool UI.
3. Enter password, but pause or cover the screen while typing.
4. Click `Run New Emails`.
5. Show these messages:

```text
Skipped non-CV attachment: ...
CV attachments found: ...
New candidates added: ...
Excel file: ...
```

6. Open the Excel tracker.
7. Show headings and sorting.
8. Blur or hide:

```text
Email
Contact Number
CV File path
Resume Hash
```

## Recommended Tools

- Microsoft Clipchamp
- PowerPoint screen recording
- Teams recording
- Windows Snipping Tool video recording
