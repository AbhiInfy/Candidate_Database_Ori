# Hiring Automation Tool

Client-facing deliverables for Motifzone Pvt Ltd.

## Files

- `Hiring Automation Tool - Client Presentation.pptx`
  - PowerPoint presentation for client walkthrough.

- `Hiring Automation Tool - Video Presentation.html`
  - Browser-based video-style presentation with voice narration using the browser speech engine.
  - Open in Microsoft Edge or Chrome and click `Play With Voice`.

- `Hiring Automation Tool - Narration.wav`
  - Standalone voiceover audio file.

- `Hiring Automation Tool - Voiceover Script.txt`
  - Narration script used for the presentation.

- `Run Hiring Automation Tool.bat`
  - Client launcher for the tool UI.

- `How To Record Live Demo Video.md`
  - Guide for recording the actual tool running on your system.

- `Hiring_Automation_Candidates.xlsx`
  - Sample/output candidate tracker.

## Recommended Demo Flow

1. Open `Hiring Automation Tool - Client Presentation.pptx`.
2. Open `Hiring Automation Tool - Video Presentation.html`.
3. Click `Play With Voice`.
4. Run `Run Hiring Automation Tool.bat` to show the working automation UI.
5. Follow `How To Record Live Demo Video.md` if you want to record the actual live process.

## Note

The HTML presentation is the voice-enabled presentation. If an MP4 is required, open the HTML or PPT and record/export using PowerPoint, Clipchamp, Teams recording, or any screen recorder.
